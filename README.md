# 项目简介

基于深度学习的晶圆体缺陷检测系统，运用YOLOv10算法与自定义YOLO数据集，实现了高效准确的缺陷识别。该系统包含UI界面，便于操作者进行实时监控与结果分析。通过Python项目源码及预训练模型，系统展现出了良好的技术特点，如快速响应和易于部署，为晶圆制造领域提供了有力的技术支持。


## 核心功能
- 基于深度学习的晶圆体缺陷检测，采用YOLOv10算法
- 使用自定义YOLO数据集进行模型训练
- 提供UI界面，支持实时监控与结果分析
- 包含Python项目源码及预训练模型
- 实现高效准确的缺陷识别功能

## 适用场景
- 晶圆制造领域的质量控制与检测
- 半导体生产过程中的缺陷自动识别
- 工业生产线的实时监控系统
- 需要快速响应的缺陷检测场景

## 技术特点
- 快速响应的检测性能
- 易于部署的系统架构
- 基于Python的开源实现
- 结合深度学习与计算机视觉技术
